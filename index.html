Understanding Web & Application Servers

This website explores how web and application servers function in client-server architecture, their evolution, and current technologies.

Key Terms

Term Definition
Client-Server Architecture A model where clients request resources or services, and servers provide them.
Web Server Handles HTTP requests and serves static or dynamic web content.
Application Server Executes business logic and generates dynamic content.
HTTP Protocol used by web servers and browsers to communicate.
Static Content Files that donâ€™t change (e.g., HTML, CSS, images), served directly.
Dynamic Content Content generated in real time based on user input or data.
CGI Method for web servers to run external programs and produce dynamic content.
Java Servlet Java program that runs on a server to respond to web requests dynamically.
Reverse Proxy Forwards client requests to backend servers for processing.
Serverless Computing Runs code on-demand in the cloud without managing server infrastructure.
